# Support Case Simulator

A Python project that simulates real-world IT support and security investigation cases using sample logs, rule-based analysis, and structured case reports.

The goal is to model how an engineer reviews evidence, identifies patterns, and recommends next steps.

## Run the Case Analyzer
```
python analyzer/case_analyzer.py cases/quarantine_flagged_file/input.json outputs/quarantine_flagged_file_report.json
```
